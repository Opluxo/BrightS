# i386 平台移植计划 | i386 Porting Plan

v0.1.2.6

---

## 概述

将 BrightS 移植到 32 位 x86 平台，保持对 Intel Atom N270 等 CPU 的优化。

**最低要求**: i686 (PAE) / 64MB RAM / IDE/SATA

---

## 移植状态

| 阶段 | 内容 | 状态 |
|------|------|:----:|
| 1. 编译系统适配 | `ARCH=i386` 编译开关，`-m32` 标志 | ✅ |
| 2. PAE 分页重构 | 3 级 PAE 分页，移除 PML4 | ✅ |
| 3. 中断与异常处理 | 重写中断入口，适配 iret | ✅ |
| 4. 系统调用接口 | sysenter/sysexit，32 位调用约定 | ✅ |
| 5. 引导与初始化 | 实模式→保护模式→内核跳转 | ✅ |
| 6. 完整内核功能移植 | AHCI/NVMe/网络/用户态进程/Shell | ⏳ |

---

## 兼容性保证

- `kernel/core/`、`kernel/fs/`、`kernel/net/` 完全架构无关
- `drivers/` 无需修改
- 仅需修改 `arch/i386/` 目录
